All statewide licensees use the California Cannabis Track and Trace (CCTT) system to track the movement of cannabis and cannabis products through the supply chain. This is known as “seed to sale” tracking.

How to get a CCTT account

Only licensed cannabis businesses can use CCTT. After your license is issued, you have 10 days to:

  • Complete account manager system training
  • Request access to the system
  • Verify your account
  • Order plant and/or package tags

The CCTT software provider, Metrc, has resources and guides to help you use CCTT and information for third-party software vendors who want to set up an application programming interface (API).

Access Track and Trace

CCTT uses Metrc software. Metrc is web-based software that can be access through an internet browser, internet-connected phone or a third-party interface.
