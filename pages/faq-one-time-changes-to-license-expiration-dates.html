Licensees with a valid processor, nursery, or cultivation license can now request a one-time change to their license expiration date. 

How do I know if I qualify for this option?

To qualify for a one-time change to a license expiration date, the following requirements must be met: 

  • The eligible license must be valid,  
  • The licensee must select a new expiration date that is on or up to twelve months after the date of request, 
  • The licensee must pay any applicable license fee, and 
  • The licensee must meet all applicable renewal requirements for the license type.   

How do I make this request? 

Eligible licensees can submit a request to make one-time change to their license expiration date during license renewal or at any time while the license is valid using the Cannabis Licensing System (CLS). 

View steps for using the CLS

How will fees be calculated? 

When a licensee with a valid processor, nursery, or cultivation license makes a one-time change to the license expiration date, DCC will calculate any credit the licensee is eligible for using a Prorated Daily License Fee.  The credited amount will be automatically applied toward the license fee associated with the Modified License Term.
